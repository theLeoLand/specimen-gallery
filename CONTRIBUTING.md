# Contributing to Specimen Gallery

Thanks for your interest in contributing! Specimen Gallery is an open-source, community-built collection of high-quality specimen images.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Set up the development environment (see README.md)
4. Create a new branch for your work

## How to Contribute

### Reporting Bugs

- Check existing issues first to avoid duplicates
- Use a clear, descriptive title
- Include steps to reproduce the bug
- Include your environment details (Ruby version, OS, browser)

### Suggesting Features

- Open an issue first to discuss the idea
- Explain the use case and why it would benefit users
- Be open to feedback and iteration

### Submitting Code

1. **Small PRs are better** — easier to review and merge
2. **Open an issue first** for large changes
3. **Follow existing code style** — the codebase uses standard Rails conventions
4. **Write meaningful commit messages**
5. **Test your changes** — run `bin/rails test` before submitting

### Pull Request Process

1. Update documentation if needed
2. Ensure all tests pass
3. Request a review
4. Be responsive to feedback

## Code Style

- Ruby: Follow standard Rails conventions
- Views: Use Tailwind CSS classes, keep ERB clean
- JavaScript: Stimulus controllers, vanilla JS preferred

## Community Guidelines

- Be respectful and constructive
- Assume good intentions
- Help others learn

## Questions?

Open an issue or reach out — we're happy to help newcomers get started.
